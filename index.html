Event Handling?

Event handling means making your webpage react to these events.
You attach a “listener” to an element, so when the event happens,
some action runs in response. For example, clicking a button might show
a message, or typing in a form might trigger validation.

How it works

  1. A user interacts with the page (click, type, scroll, etc.).
  2. The browser detects this action as an event.
  3. If you’ve set up a listener, your JavaScript code runs and responds.

Different Ways to Handle Events

  • Inline handling: putting the action directly in the HTML (not recommended, messy).
  • Property handling: assigning one action to an element’s event property (works, but limited).
  • Event listeners: the modern way, where you attach one or multiple handlers cleanly from JavaScript.
